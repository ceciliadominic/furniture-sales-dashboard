# 🪑 Furniture Sales Dashboard Analysis

## 📊 Project Overview
This project analyzes sales performance within a furniture business using Microsoft Excel. The dashboard provides insights into revenue, profitability, regional performance, and product-level trends to support strategic decision-making.

## 🎯 Objectives
- Evaluate overall sales and profit performance
- Identify top-performing categories and regions
- Detect loss-making products and inefficiencies
- Provide actionable insights for business growth

## 🛠️ Tools Used
- Microsoft Excel
  - Data Cleaning
  - Pivot Tables
  - Interactive Dashboard
  - Charts & Visualizations

## 📈 Key Insights
- Generated over **$742K in total sales**, but only **$18.4K in profit**, indicating low profit margins
- **Furniture** is the top-performing category by sales
- The **West region** leads in revenue generation
- **Tables** sub-category is the biggest loss-maker, negatively impacting profitability
- Profitability varies significantly across products despite strong sales performance
